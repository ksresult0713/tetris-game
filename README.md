# Tetris Game

A polished, dependency-free browser Tetris built with HTML, CSS, and Canvas.

## Play

Open `index.html` in a browser, or serve the folder with any static file server.

## Controls

- Arrow Left / Arrow Right: move
- Arrow Down: soft drop
- Arrow Up or X: rotate right
- Z: rotate left
- Space: hard drop
- C: hold
- P: pause
- Enter: restart after game over

## Features

- Seven-bag piece randomizer
- Hold and next piece preview
- Ghost piece
- Score, level, line count, and local best score
- Responsive touch controls
